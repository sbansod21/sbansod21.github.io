Cathy Cash- [SheInnovates 2020]

| Devpost Submission |

Cathy cash is a money managing app targetted for students at the University of Pittsburgh. It has features such as targetted spending on specific amenities, rewards for meeting set targets as well as connection to the student account at Pitt to use university funds such as Panther funds and Dining dollars. This program was done using MIT App Inventor. My contribution to this project included helping with the back-end algorithm and the user interface of the app.
