Jukebox - [SheInnovates 2021]

| Github |

Jukebox is a study tool that can be used to increase focus. Studies show that certain music allows students to perform better in certain tasks. We used the spoify API which allowed us to measure the "dancability" of each song. We then divided the songs into certan sections. Our program allows the user to pick the activity that they are doing, and music is played based on our research. This program was done using Python and HTML. I contributed to the back end of the program as well as the spotify API.
